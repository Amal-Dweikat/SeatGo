
# 🚗 SeatGo – Ride Sharing Mobile App

## 📌 Overview

SeatGo is a mobile ride-sharing application built with React Native that connects drivers with available seats to passengers traveling in the same direction.

The app focuses on optimizing commuting by reducing costs, improving ride efficiency, and enabling flexible trip scheduling.

---

## ✨ Key Features

* 🔐 JWT-based authentication (Login, Register, Reset Password)
* 🚘 Trip creation (one-time & recurring trips)
* 🔍 Advanced trip search with filters (location, time, price)
* 📅 Booking system with request/accept/reject flow
* ⭐ Two-way rating system (driver ↔ passenger)
* 🔔 Smart notifications (reminders, updates, cancellations)
* ❤️ Favorite drivers with alerts
* 📍 Suggested trips based on user location
* 💬 Temporary trip-based chat system
* 🛠️ Admin dashboard for managing users and trips

---

## 🧠 Technical Highlights

* Built with **React Native** for cross-platform mobile development
* Backend powered by **Laravel REST API**
* Secure authentication using **JWT tokens**
* Designed scalable booking logic with time-based request handling (4h / 24h rules)
* Real-time-like interaction through notifications and trip-based chat system

---

## 🧰 Tech Stack

* Frontend: React Native
* Backend: Laravel
* Authentication: JWT
* Database: MySQL
