SeatGo is a ride-sharing application that helps people who are going in the same direction share a ride together. 
The main idea is to connect drivers who have empty seats with passengers traveling along the same route.
 The app allows users to create or search for rides based on location, destination, and time.
 Drivers can offer available seats, while passengers can easily find rides that match their route. SeatGo
provides an alternative for people who prefer not to use traditional transportation or want a more flexible and efficient option instead of
regular commuting. It also aims to reduce transportation costs, decrease traffic congestion, and make commuting more efficient.
